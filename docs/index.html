Features that Feel Like Magic

Built-in tools that normally cost hundreds of dollars a month, now running on your terminal.

🌐

Multi-CDN Uploader

Never worry about Google Drive bandwidth limits again. Upload local Markdown images directly to ImgBB, Cloudinary, or GitHub automatically.

🔄

2-Way Sync (Pull & Push)

Already have 100 posts on Blogger? Run pull to download them all as Markdown files, edit them locally, and publish them back instantly.

🤖

AI Agent Ready

Comes with built-in AGENT.md and Skills. Tell your AI to "run autoblogging skill" and watch it generate and publish content on autopilot.

⚡

Smart Hashing

Saves your API quota by hashing your Markdown files. If you don't change the content, it skips uploading. Instant deployment for thousands of files.
