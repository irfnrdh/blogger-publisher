Enterprise-Grade Features

Everything you need to turn a Blogger site into a massive, AI-managed content empire.

🤖

MCP Server (25 Tools)

Register mcp-blogger-server in any MCP-compatible AI client and manage your entire blog through natural language — no terminal needed.

🏗️

Interactive TUI & Scaffolding

Run blogger-publisher with no arguments for a guided, colorful interactive menu. Init Workspace creates folders, config, and sample articles automatically.

🚀

Multi-CDN Image Routing

Upload local images automatically to ImgBB, Cloudinary, or GitHub jsDelivr — no more Google Drive bandwidth limits.

🔄

2-Way Sync (Pull & Push)

Already have 100 posts on Blogger? Run pull to download them all as Markdown, edit locally, then publish them back instantly.

🌐

Global Config (No More .env)

Set up once in ~/.blogger-publisher/config.json. Publish from any folder on your machine without re-configuring.

⚡

Smart Hashing, Zod & SEO

Save API quota with content-hash change detection. Strict Zod frontmatter validation. Custom URL slugs & meta descriptions supported.
