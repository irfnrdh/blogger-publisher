Enterprise-Grade Features

Everything you need to turn a simple Blogger site into a massive content empire.

🤖

MCP Server (25 Tools)

Register mcp-blogger-server di AI client Anda (Antigravity, Claude, Cursor) dan kelola seluruh blog via percakapan natural — tanpa sentuh terminal sama sekali.

🏗️

Interactive TUI & Scaffolding

Jalankan blogger-publisher tanpa argumen dan nikmati menu interaktif berwarna. Init Workspace membuatkan folder, config, dan contoh artikel otomatis.

🚀

Multi-CDN Image Routing

Upload gambar lokal langsung ke ImgBB, Cloudinary, atau GitHub jsDelivr secara otomatis — tanpa khawatir bandwidth limit Google Drive.

🔄

2-Way Sync (Pull & Push)

Sudah punya 100 post di Blogger? Jalankan pull untuk download semuanya jadi Markdown, edit lokal, lalu publish kembali seketika.

🌐

Global Config (Bebas .env)

Simpan kredensial sekali di ~/.blogger-publisher/config.json. Publish dari folder mana pun di komputer Anda tanpa setup ulang.

⚡

Smart Hashing, Zod & SEO

Hemat API quota dengan deteksi perubahan berbasis hash. Validasi frontmatter ketat dengan Zod. Mendukung Custom URL Slug & Meta Description.
