# Security Policy

## Supported Versions

Only the latest major version is currently being supported with security updates.

| Version | Supported          |
| ------- | ------------------ |
| 1.x.x   | :white_check_mark: |
| < 1.3.4 | :x:                |

## Reporting a Vulnerability

Please do not report security vulnerabilities through public GitHub issues.

If you believe you have found a security vulnerability in Blogger Publisher, please report it to kang.fand@gmail.com.

We will acknowledge receipt of your vulnerability report and strive to send you regular updates about our progress. If you have not received a reply to your email within 48 hours, please follow up to ensure we received your original message.
